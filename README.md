# BrainBloom – AI Flashcard System

BrainBloom is an AI-powered flashcard application designed to help users learn efficiently using spaced repetition and interactive quizzes.

## 🚀 Features
- AI-generated flashcards using OpenAI API
- Quiz mode to test knowledge
- Spaced repetition for better memory retention
- User performance tracking
- Star rating system for flashcards

## 🛠️ Tech Stack
- Python (Flask)
- SQL Server
- HTML/CSS/JavaScript
- OpenAI API


## ⚙️ How to Run
1. Clone the repository
2. Install required libraries
3. Run the Flask app
4. Open in browser
